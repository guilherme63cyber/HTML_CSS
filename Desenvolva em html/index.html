Conteúdo

As tags h1 a h6 no html são usadas para definir cabeçalhos ou titulos em uma pagina. Elas represemtam diferentes niveis de hierarquia, onde h1 é o nivel mais alto(titulo principal) e h6 é o nivel mais baixo

<h1>

Usado para o titulo principal da pagina

É o nivel mais alto na hierarquia Titulos


<h2>

Usado para subtitulos ou seções importantes

É o segundo na hierarquia de Titulos


<h3>

Usado para titulos de subseções

É o terceiro na hierarquia de Titulos


<h4>

Usado para titulos dentro de subseções

É o quarto na hierarquia de Titulos


<h5>

Usado para titulos menores ou detalhes especificos

É o quinto na hierarquia de Titulos


<h6>

Ultilizado para titulos de menor importancia ou detalhes mais especificos

É o sexto na hierarquia de Titulos


Historia do html:


Criação:

O HTML foi proposto e criado por Tim Berners-Lee, um cientista da computação britânico, em 1991, enquanto trabalhava no CERN (Organização Europeia para a pesquisa Nuclear). A primeira especificação formal do HTML, chamada HTMLN 2.0, foi lançada em 1995 pelo Word Wide Web Consortium (W3C).


Versões Principais do HTML:


HTML 2.0(1995)

A primeira versão formal do HTML, introduzindo muitos dos elementos básicos.


HTML 3.2 (1997)

Adição de suporte para trabalhos e applets Java.

esta versão também viu o uso mais difundido dos navegadores.


HTML 4.01(1999)

Introdução de folhas de estilo em cascata (CSS).

Adiçao de Suporte para Script do lado do cliente (Java Script)

Divisão em três variantes: Stike, Trasitional e Frameset.


HTML 5(2014)

Uma Revisao significativa do HTML, trazendo novos recursos e melhorias.

Suporte Nativo para áudio e video

Introdução de novas tags semânticas como <article>,<section>,<header>,<footer>.
